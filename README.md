# spring-boot-task-manager

## 🚀 Live API Documentation
The API is deployed on Render. You can explore and test the endpoints directly through the Swagger UI:

[👉 View Swagger API Docs](https://spring-boot-task-manager.onrender.com/swagger-ui/index.html)

### How to use:
1. Open the link above.
2. Expand any endpoint (e.g., `POST /users`).
3. Click **Try it out**, enter your data, and click **Execute**.
